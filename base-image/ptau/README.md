
Powers of tau files
===================

These are universal reference string for pairing based SNARK-s (KZG, Groth16, etc) using the bn128 elliptic curve.

Different sizes are available:

- 2^8 (380 kB)
- 2^12 (4.8 MB
- 2^16 (75 MB)
- 2^20 (1.2 GB)
- 2^22 (4.8 GB)
